grakel.random_walk¶

class grakel.random_walk(executor=<function <lambda>>, normalize=False, verbose=False, lamda=0.1, method_type='fast', kernel_type='geometric', p=None)[source][source]¶

The random walk kernel class.

See [KTI03], [GartnerFW03] and [VBS06].

Parameters:

lambda : float

A lambda factor concerning summation.

method_type : str, valid_values={“baseline”, “fast”}

The method to use for calculating random walk kernel:
  • “baseline” Complexity: \(O(|V|^6)\) (see [KTI03], [GartnerFW03])
  • “fast” Complexity: \(O((|E|+|V|)|V||M|)\) (see [VBS06])

kernel_type : str, valid_values={“geometric”, “exponential”}

Defines how inner summation will be applied.

p : int or None

If initialised defines the number of steps.

pairwise_operation(X, Y)[source][source]¶

Calculate the random walk kernel.

Fast: Spectral demoposition algorithm as presented in [VBS06] p.13, s.4.4, with complexity of \(O((|E|+|V|)|E||V|^2)\) for graphs witout labels.

Baseline: Algorithm presented in [KTI03], [GartnerFW03] with complexity of \(O(|V|^6)\)

Parameters:

X, Y : Objects

Objects as produced from parse_input.
Returns:

The kernel value.
